# Spatial Hub Scotland (data-spatialhub-scot)

Spatial Hub Scotland is a regional government open-data portal for United Kingdom running **CKAN**, exposing a standardized catalog API over ~496 datasets.

**URL:** [Visit APIs.json URL](https://raw.githubusercontent.com/api-evangelist/data-spatialhub-scot/refs/heads/main/apis.yml)

**Run:** [Capabilities Using Naftiko](https://github.com/naftiko/fleet?utm_source=api-evangelist&utm_medium=readme&utm_campaign=data-spatialhub-scot-api-evangelist&utm_content=repo)

## Type
- **kind:** government  ·  **software:** CKAN  ·  **version:** 2.10.3

## API
- **Spatial Hub Scotland CKAN Action API** — base `https://data.spatialhub.scot/api/3/action`. [Docs](https://docs.ckan.org/en/latest/api/) · [Portal](https://data.spatialhub.scot)
- Live endpoint verified: `https://data.spatialhub.scot/api/3/action/status_show` · ~496 datasets
